# Sanskruti Jagdale - Portfolio Website

A luxury, high-end portfolio website designed with a focus on elegance, minimalism, and editorial typography.

## 🌟 Features

- **Luxury Aesthetic**: Soft glassmorphism, refined typography (Playfair Display & Outfit), and pastel gradients.
- **Interactive Elements**: Typewriter effect in the hero section, smooth scroll reveal animations, and hover effects.
- **Responsive Design**: Fully responsive layout that looks great on all devices.
- **Custom Assets**: Abstract SVG backgrounds generated for project thumbnails.

## 📁 File Structure

- `index.html`: Main content structure.
- `style.css`: All styling, including animations and glassmorphism utilities.
- `script.js`: interactive logic (typewriter, scroll reveal).
- `assets/`: Contains SVG background images for projects.

## 🛠️ How to Customize

1.  **Resume**: Replace `Sanskruti_Resume_Updated.pdf` with your actual resume file in the root directory, or update the links in `index.html`.
2.  **Links**: Update the `#` in the `href` attributes for LinkedIn and GitHub in the footer.
3.  **Images**: You can replace the SVG files in `assets/` with actual project screenshots.
4.  **Text**: Edit `index.html` to update your bio, projects, or skills.

## 🚀 Running the Site

Simply open `index.html` in any modern web browser.
